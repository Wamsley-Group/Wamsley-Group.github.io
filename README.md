# Wamsley-Group.github.io

Website for Wamsley Group - A real estate website with contact forms that work on both Netlify and GitHub Pages.

## Form Handling

This website uses **FormSubmit.co** for form submissions, which works on:
- ✅ GitHub Pages (with custom domain support)
- ✅ Netlify
- ✅ Any static hosting platform

See [GITHUB_PAGES_FORMS_SETUP.md](GITHUB_PAGES_FORMS_SETUP.md) for complete setup instructions.

## Quick Start

1. Deploy to GitHub Pages or Netlify
2. Verify email address on first form submission (one-time only)
3. Forms will work automatically after verification

All forms send notifications to: `pipoat@mail.uc.edu`
