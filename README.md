## Quick Start

### Deployment

The site is automatically deployed via GitHub Pages from the `main` branch.

**Live site:** https://wamsleygroup.com

### Form Setup

Forms use **FormSubmit.co** and work on any static hosting platform:
1. Deploy the site
2. Submit a test form
3. Verify email address (one-time only)
4. Forms will work automatically
